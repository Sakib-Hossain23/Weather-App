# Weather App

This app made with HTML, CSS, and JavaScript to deliver a user-friendly interface and fetches data using the OpenWeatherMap API.

# Features

- **Live Weather Data**
- **5 Days Forecast**
- **Explore Section**
- **Add More Citys**

# Technologies Used:
HTML
CSS
JavaScript
